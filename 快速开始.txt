旅游计划生成器 - 快速开始指南
====================================

步骤1：打开命令行
------------------
- 按 Win + R，输入 cmd 或 powershell，按回车
- 或者直接在 travel-planner 文件夹中，按住 Shift + 右键，选择"在此处打开PowerShell窗口"

步骤2：安装依赖包
------------------
在命令行中输入：
    pip install -r requirements.txt

如果提示权限错误，可以尝试：
    pip install --user -r requirements.txt

步骤3：运行程序
------------------
在命令行中输入：
    python app.py

看到类似以下输出说明启动成功：
    * Running on http://127.0.0.1:5000
    * Running on http://0.0.0.0:5000

步骤4：打开浏览器
------------------
在浏览器中访问：
    http://localhost:5000
    或
    http://127.0.0.1:5000

步骤5：使用应用
------------------
1. 填写旅游天数（必填）
2. 填写目的地（必填）
3. 可选：填写预算和兴趣偏好
4. 点击"生成旅游计划"按钮
5. 等待几秒钟，AI会生成详细的旅游计划

常见问题：
----------
Q: 提示找不到模块？
A: 运行 pip install flask flask-cors openai python-dotenv

Q: 提示端口被占用？
A: 修改 app.py 最后一行，将 port=5000 改为其他端口，如 port=5001

Q: 生成计划失败？
A: 检查网络连接，确保API密钥正确

Q: 如何停止程序？
A: 在命令行中按 Ctrl + C
